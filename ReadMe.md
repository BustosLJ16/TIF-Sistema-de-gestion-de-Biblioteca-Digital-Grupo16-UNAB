## 🎯 Objetivo
Desarrollar un sistema de gestión bibliotecaria que permita administrar libros, usuarios, bibliotecarios y préstamos, aplicando conceptos avanzados de programación orientada a objetos en Python.

## ℹ️ Información general sobre el Proyecto:
Este proyecto fue desarrollado utilizando el lenguaje de programación *Python*, aplicando los principios fundamentales de la Programación Orientada a Objetos (POO).
En el mismo, nos centramos en imitar el funcionamiento del Sistema de una Biblioteca, con algunas de sus funciones principales, tales como: 
- Manejo de los datos de libros (Alta, Modificación, Baja y Listado)
- Manejo de usuarios con roles diferenciados (Usuario y Bibliotecario).
- Gestión de Prestamos y Devoluciones de uno o más Libros.

## 🚀 Características Técnicas
Durante el desarrollo del proyecto se aplicaron diversos conceptos avanzados de Python:
- Programación Orientada a Objetos (POO).
- Herencia entre clases Persona, Usuario y Bibliotecario.
- Polimorfismo mediante el método `mostrar_info()`.
- Metaclase personalizada para garantizar la implementación de métodos obligatorios.
- Decoradores para personalización de listados.
- Patrón de diseño Singleton para la gestión centralizada de la Biblioteca.
- Gestión automática de fechas mediante el módulo `datetime`.
- Menús interactivos para Usuarios y Bibliotecarios.
- Validaciones para evitar registros duplicados de libros, usuarios y bibliotecarios.
