## ℹ️ Información sobre el Proyecto:
En este Proyecto fue construido bajo el Lenguaje de Programación *Python*, basado en la Programación Orientada a Objetos (POO), con sus pilares fundamentales.
En el mismo, nos centramos en imitar el funcionamiento del Sistema de una Biblioteca, con algunas de sus funciones principales, tales como: 
- Manejo de los datos de libros (Alta, Modificación, Baja y Listado)
- Manejo de Usuarios y Roles como Cliente y Bibliotecario.
- Gestión de Prestamos y Devoluciones de uno o más Libros.

Entre otras funciones varias que se verán a lo largo del código.
